# Changelog

## v0.7.5 - 2026-05-04

- 新增可选 `Ctrl+[` 模式：在快捷键设置里开启后，`Ctrl+[` 会发送真实 Esc 并切换到英文输入法。
- “Esc 生效的应用”新增批量管理窗口，支持搜索、勾选多个应用、全选/取消当前搜索结果。
- 修复 JetBrains IDE 中双击 Shift 被单击 Shift 切换逻辑干扰的问题。

## v0.7.4 - 2026-05-04

- 修复 GitHub Release 构建产物版本号显示为 `${GITHUB_REF#refs/tags/v}` 的问题。
- 更新检查改用新的 `makerjackie/macvimswitch` 仓库地址。
- "忽略此版本" 现在会在自动和手动检查更新时生效。
- 调整 Release 安装说明：升级时优先直接替换 app，只有权限失效时再重新授权。

## v0.7.3 - 2026-05-04

- 修复偶发的输入法延迟问题：菜单栏看起来已经切换，但实际打字还停留在旧输入法。
- 输入法切换现在会等待系统确认，并对较慢的 CJKV 输入法做重试。
- 避免单击 Shift 同时被旧逻辑和快捷键管理器处理，减少反向切换和重复切换。
- `jk` 序列继续使用原有的时间窗口判断，避免两套缓冲状态互相干扰。
- 本地构建和 GitHub Release 构建都加入新的快捷键管理文件。
